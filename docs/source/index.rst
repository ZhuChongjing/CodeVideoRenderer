CodeVideoRenderer
=================

告别枯燥录屏。利用 Python 核心模块，生成带有智能运镜、语法高亮和丝滑动画的专业视频。

:class:`~.CameraFollowCursorCV` 是本库的核心类，专门设计用于模拟代码输入的动画过程。它通过逐行、逐字符的动画效果，配合平滑的摄像机跟随，创造出具有专业水准的代码演示视频。
