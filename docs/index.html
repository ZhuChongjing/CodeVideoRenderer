CodeVideoRenderer

CodeVideoRenderer
智能相机跟随动画

告别枯燥录屏。利用核心模块 CameraFollowCursorCV，将您的代码演示提升至电影级的专业水准。

探索核心技术

一切，聚焦于你

光标移动，摄像机即动。平滑的缩放和跟随，只为呈现最佳的演示视角。

demo.py 
from CodeVideoRenderer import CameraFollowCursorCV
video = CameraFollowCursorCV(
    code_string="print('Hello, World!')",
    language="python"
)
video.render()

精确可控的渲染引擎

相机缩放

当光标遇到长行代码时，摄像机会动态缩小以保持视野。

打字间隔

自定义打字间隔的随机范围。避免机械式输入，增强真实感。

行间距

自定义代码行之间的垂直间距，精确控制代码文本的排版密度。

代码输入

支持直接输入代码字符串或读取文件。

进度反馈

渲染时在终端实时显示基于 rich 的进度条、渲染耗时和详细摘要。

Manim 核心

基于强大的 Manim 动画引擎，保证输出视频的极高画质和渲染的稳定性。
