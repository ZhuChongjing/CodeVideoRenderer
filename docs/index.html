快速开始

5分钟上手，从安装到生成第一个视频

1 安装依赖库

pip install CodeVideoRenderer

要求 Python 3.9+ 版本

2 编写渲染脚本 

from CodeVideoRenderer import *
video = CodeVideo(code_string="print('Hello World!')", language='python')
video.render()

可通过创建多个CodeVideo对象，实现批量生成
