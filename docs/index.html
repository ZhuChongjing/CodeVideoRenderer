CodeVideoRenderer

v1.0.9.post2 现在可用

让代码演示
极具电影感

告别枯燥录屏。利用 Python 核心模块，生成带有智能运镜、语法高亮和丝滑动画的专业视频。

探索核心技术 pip install CodeVideoRenderer

一切，聚焦于你

光标移动，摄像机即动。平滑的缩放和跟随，只为呈现最佳的演示视角。

demo.py 
from CodeVideoRenderer import CameraFollowCursorCV
video = CameraFollowCursorCV(
    code_string="print('Hello, World!')",
    language="python"
)
video.render()

精确可控的渲染引擎

相机缩放

当光标遇到长行代码时，摄像机会动态缩小以保持视野，宛如电影运镜。

拟人输入

自定义打字间隔的随机范围。避免机械式输入，增强真实感。

排版控制

自定义代码行之间的垂直间距，精确控制代码文本的排版密度。

灵活输入

支持直接输入代码字符串或读取文件，无缝集成到你的工作流。

实时反馈

渲染时在终端实时显示基于 rich 的进度条、耗时统计和详细摘要。

强大核心

基于强大的 $\mathbb{M}\text{anim}$ 动画引擎，保证输出视频的超高画质和极高的稳定性。
